# LeaveWise

Smart Attendance & Leave Planning App

Project Overview:

LeaveWise is a smart mobile application designed to help students plan and manage their attendance efficiently. The application allows students to upload their timetable, plan attendance for each lecture slot, and later mark actual attendance. The system compares planned and actual attendance and generates warnings if attendance falls below the minimum required percentage.
This project focuses on improving attendance planning, reducing shortage risks, and helping students make better academic decisions.

Objectives:

1) To help students plan semester attendance in advance
2) To allow students to track actual attendance
3) To provide subject-wise and overall attendance calculation
4) To generate automatic warnings for low attendance
5) To allow timetable and lecture time slot customization
6) To improve student attendance management through smart planning
